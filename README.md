Sleep Recorder:

Startup:

1. install all dependencies in requirements.txt of both gui and backend:

  pip install -r requirements.txt
  (for backend only)

3. pull up 2 cli's, one in gui, one in backend

4. run in backend:

source .venv/bin/activate

4. run in gui:

npm start

5. run in backend:

python app.py

6. go to localhost:8080 and use
